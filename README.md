大家好，欢迎来到我的代码仓库，该仓库自知识星球短链接项目改编，但是加入了很多我自己独立的开发的元素，全程手搓，没有拉取代码，如有建议请指导谢谢！

##首先这个项目主要的业务就是将长链接变成长连接来进行跳转，经过我个人实践，已经上线了云服务器(2核4gb)，其中短链接跳转那边需要统计用户数据，所以涉及到多张表的增删改查，在此我做了mq异步落库处理，并采用布隆过滤
器和缓存空值来防止缓存穿透，分布式锁防击穿等。还有就是我没有用sentinel控制台监控，内存原因，我自己写了个mq异步落库的监控。
但问题是我的服务器性能不太好，因此我做了服务降级处理，默认不开启mq，而是stream流。由于stream流过大可能会导致我内存溢出所以我做了trim来裁剪流的长度，我做了redis的zset+lua脚本来检测短链接跳转的qps以此来达到
动态调整裁剪流的速率的效果。

##目前系统稳定运行，短链接跳转接口稳定54qps，毫无压力，欢迎访问我的网站：https://pengwater.xin
